**YouTube Title:** NeuronScope — Real-Time LLM Activation Sparsity Visualization | Puentes Demo

**YouTube Description:**
NeuronScope is a research tool built at Boston Children's Hospital / Harvard Medical School that visualizes activation sparsity in large language models in real time. It runs inference on ActivSparse-SLM — a custom 125M parameter sparse language model combining SLTrain, ReLU activations, DeepSeekMoE, and Wanda post-training pruning — achieving 50% FFN weight sparsity with only 0.3% perplexity degradation across 9 training runs. Built with FastAPI + WebSocket + Canvas 2D. Built as a Puentes fellowship application demo.
