# Inbox

**Capture zone.** Drop everything here first — process during weekly review.

## How to Use

1. Create a new note here for any idea, task, link, or thought
2. Don't worry about formatting — get it out of your head
3. During weekly review: file each item to the correct folder or delete it
4. Goal: inbox zero by end of each week

## Processing Guide

When processing an item, ask:
- Is it **actionable?** → Add to a project note or task system
- Is it **reference?** → File to `050-Knowledge/` or relevant project
- Is it **system-related?** → File to `010-System/` or `020-Infrastructure/`
- Is it **life-related?** → File to `060-Life/`
- Does it **belong to a project?** → File to `030-Projects/<project>/`
- Is it **not useful?** → Delete it
